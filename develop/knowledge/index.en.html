Язык представлений знаний OWL

Экспертная система использует Базу Знаний. База Знаний представлена в виде XML файлов - фреймов. Существует несколько различных типов фреймов. Экспертная система работает с фреймами знаний структуры OWL и RuleML.

OWL — (англ. Web Ontology Language) — язык онтологии для интернета на основе XML/Web стандарта. Язык веб-онтологий OWL призван обеспечить язык, который может быть использован для описания классов и отношений между ними, которые присущи для веб-документов и приложений.
В основе языка — представление действительности в модели данных объект — свойство. По признанию создателей, owl пригоден не только для описания web страниц, но и любых объектов действительности. Каждому элементу описания в этом языке ставится в соответствие www адрес, но во всем остальном — это чудесное описание механизма reference системы на базе модели объект — свойство.
