Репозиторий AReason

Уже около года я занимаюсь причесыванием исходников проекта AReason. Привел все более или менее к одному виду. Скомпилировал около 90% программулек, которые делал (вернее пытался делать) в рамках проекта AReason.

Проект AReason - это система из множества отдельных программок. Каждая программа в системе, по идее, должна нести какую-то небольшую функциональность. Все программки, по идее, должны работать с единой БЗ.

К сожалению довести до презентабельного состояния не получается. Но я подготовил и начал выкладывать исходники проекта AReason.

Страничка проекта: sourceforge.net/projects/areason
Репозиторий (Git): sourceforge.net/projects/areason/code

Склонировать репозиторий можно такой командой: 

git clone git://git.code.sf.net/p/areason/code areason-code

В Ubuntu потребуется заранее установить пакет git, в Windows нужно установить mSysGit, а так же удобную программу TortoiseGit (описание).

Для сборки потребуется Delphi (начиная с Delphi 7 и более новая). Под Linux пока собирать не пробовал. Для того, чтобы собрать в Linux нужно будет перевести исходники в Lazarus. Но это не так просто. Проблемы возникают с компонентами форм, с русскими комментариями в тексте и прочим.
