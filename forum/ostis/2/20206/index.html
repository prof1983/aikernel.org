Среда разработки для языка SCP (scpdev) Сообщение LazurkinDA » 15 ноя 2010, 21:44 Сделан релиз SCPDEV версии 0.1.0 - плагина для Eclipse, который облегчает программирование на SCP. Основные возможности: * Создание проекта * Автоматическая сборка и пересборка репозитория на лету * Подсветка синтаксиса при редактировании scs/scsy/m4scp * Указание ошибок в редакторе после неуспешной компиляции * Быстрый просмотр сгенерированного кода scs для m4scp-файла * Поддержка всех аргументов start-pm из пользователького интерфейса * Готовые шаблоны для более быстрого набора текстов m4scp Скачать можно тут (https://sourceforge.net/projects/ostis/files/) в разделе tools.
