Установка модуля sc-core Сообщение LazurkinDA » 16 мар 2010, 18:51 Модуль sc-core - набор библиотек и программ, которые обеспечивают нижний уровень поддержки sc-систем. В него входят следующие библиотеки: [ul] [*]libTGF - библиотека для работы с бинарным представлением графов; [*]libSys - библиотека обеспечения переносимости на другие ОС; [*]libSC - библиотека моделирование sc-памяти; [*]libPM - библиотека scp-интерпретатора и навигационно-поисковой машины. [/ul] Адрес sc-core в svn (https://ostis.svn.sourceforge.net/svnroot/ostis/trunk/kpm/sc-core/) Пока есть возможность установки только из исходных текстов. Описание процесса установки sc-core (http://ostis.net/wiki/%D0%A3%D1%81%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BA%D0%B0_%D0%B1%D0%B8%D0%B1%D0%BB%D0%B8%D0%BE%D1%82%D0%B5%D0%BA%D0%B8_%D0%BC%D0%BE%D0%B4%D0%B5%D0%BB%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F_sc-%D0%BF%D0%B0%D0%BC%D1%8F%D1%82%D0%B8) Пример использования библиотеки libSC (https://ostis.svn.sourceforge.net/svnroot/ostis/trunk/kpm/sc-core/examples/wave_find_path/) Ожидаются вопросы и предложения.
