Описание

Задание «Хранение настроек приложения»

Необходимо спроектировать иерархию классов, с помощью которых возможно хранить настройки приложения. Настройки представляют собой набор именованных разнотипных параметров (типов логический, строка, целое число, дата и другие). Базовый класс должен быть независим от выбора хранилища настроек. Потомки реализуют работу с конкретным хранилищем. Архитектура должна быть выполнена таким образом, чтобы позволять использовать в качестве хранилища базу данных, INI – файл, XML – файл.

Реализовать указанную иерархию для БД. Данные должны быть нормализованы, хранение разнотипных данных в одинаковых структурах не допускается.

Реализовать тестовую форму с использованием реализованных структур.
