Разработчикам

Проект решено сделать коммерческим. Каждый модуль оформлять как отдельную программу и продавать их по отдельности.

Все модули должны работать с одинаковыми типами данных. Данные внутри программы передаются в виде XML структур. Хранение данных организуется в XML файлах. Для хранения текста предпочтительно использование OWL. Для хранения последовательности действий приедпочтительно использования языка AR.
