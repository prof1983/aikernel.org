Мой проект

Мой проект реализации искусственного разума. Находится в разработке.

Решился выложить на общий доступ свои модули AI на Delphi. В них я попытался создать базовые функции для работы с Базой Знаний (Source), Предметной Областью (Domain), Деревом Ассоциаций (Tree), Списком Решений (Results).

Эти модули выложены пока без подробного описания и без примеров применения. В скором времени я попытаюсь написать несколько наглядных программ для демонстрации принципов работы этих модулей. А пока этот раздел, я думаю, будет не интересен и не понятен.
