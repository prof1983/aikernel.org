TAIFreimType

Тип фрейма

TAIFreimType = type UInt032

Описание
Размер: 4 байта

Значения
1 - фрейм-слово
2 - фрейм-предложение
3 - фрейм-действие
-1 - Объект (предмет,существо). Стол, стул, Вася.
-2 - Связь объектов
-3 - Действие. Есть, мыть.
-4 - Свойство. Красное, хрупкое. В блоке Data находятся данные:
  1) Тип данных
  2) Сами данные
---$0201; Базовые действия для взаимодействия с внешней средой
-$FF01 - строка символов (текст) - введено временно
-$FF02 - строка символов (слово)
-$FF03 - фрейм-буква
-$FF04 - фрейм-слово
-$FF05 - фрейм-смысловая единица
-$FF06 - фрейм-предложение(одна или несколько смысловых единиц)
-$FF07 - фрейм-абзац
-$FF08 - фрейм-смысл
