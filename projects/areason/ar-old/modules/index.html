Модули

Здесь будут представлена информация об отдельных модулях: автор, описание, документация, ссылки на сайт разработчика, а так же исходные коды и/или скомпилированный модуль.

Модуль - компонент, класс, объект или библиотека (dll).

  • Базовые типы данных
    Простые типы данных для использования во всех модулях.
  • Базовые типы для AI
    Фрейм (TAIFreim), Тип фрейма (TAIFreimType), Идентификатор фрейма (TAIFreimId), Связь (TAIConnect), Данные (TAIData), Источник знаний (TAISource).
  • Базовые модули
    Базовый класс для создания модуля на Delphi (TAIModule), класс для создания модуля - контейнера модулей (TAIModules), класс для создания графического модуля - окна (TAIModuleForm).
  • Нейрон
    Класс сознания нейрона (TAINeuron). Используется для построения нейросети.
  • Нейросеть
    Класс создания нейросети (TAINeuralNetwork).
  • Поиск пути
    Исходняки программы поиска пути.
  • Искусственная жизнь
    Программа, моделирующая жизнь бактерий.

  • Объектно-ориентированный язык программирования Маркиз
    Описывается объектно-ориентированный язык программирования. Особенностью языка являются возможность определять объекты как структурированные совокупности подобъектов, конструировать их из компонент-подобъектов в процессе работы программы и изображать объекты, то есть представлять их в виде последовательности литер, идентифицирующей объект и его тип. Последнее позволяет, в частности, использовать язык для создания различных языковых процессоров (включая компиляторы, интерпретаторы, конвертеры с одного языка на другой).
