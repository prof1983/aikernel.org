TAIAssociation

TAIAssociation = record
  Handle: THandle064;
  TypeAssociation: TAIAssociationType;
  Weight: Int008;
  Reliance: UInt008;
  CountUse: UInt016;
  Use: Pointer;
end;

Описание
Размер: 16 байт - для записи на диск; 20 байт - в памяти.

Параметры
Handle: THandle064
Идентификатор фрейма с которым ассоциирован
TypeAssociation: TAIAssociationType
Тип ассоциации:
Weight: Int008
Вес связи: -128 - абсолютно не верно, 0 - не известно, 127 - абсолютно верно
Reliance: UInt008
Уверенность в весе связи: 0 - абсолютно не уверен, 255 - уверен на 99.9%.
CountUse: UInt016
Сколько записей об использовании есть TAIUse (128 записей x 16 bytes = 2k)
Use: Pointer
Указатель на место в памяти, где располагается массив записей об использовании этой ассоциации.
