Assistant

Проект Assistant создан в рамках одного большого проекта AIAssistant.

Проект создан для отработки технологий, изучения предметной области и привлечения участников к разработке.

Проект Assistant будет являеться прототипом для продукта AIAssistant.

При реализации программы Assistant предполагается выработать единое мнение о внешнем интерфейсе и фунцкциональных возможностях будующей программы AIAssistant.
Техническое задание
Версия 0.1

Техническое задание на разработку программы Assistant 0.1 Необходимо добавить следующее:

    Вкладка управления заданиями (уже реализовано)
    Все задания должны хранится в виде XML.
    Реализовать класс TTask
        ID:Integer - идентификатор
        Title:WideString - заголовок
        Comment:WideString - комментарий
        DateBegin:TDateTime - Дата и время старта реализации задания
        DateEnd:TDateTime - Дата и время окончания реализации задания
        RemindBegin=True - Напоминать при старте
        RemindEnd=True - Напоминать при окончании
        Remind=True - Периодическое напоминание
        RemindInterval - Интервал периодического напоминания, сек
    Сделать всплывающие подсказки напоминаний около Tray.
    Сохранение файла данных должно осуществляться при каждом редактировании, а не только при закрытии программы.
    Данные должны хранится в одном файле. Имя файла должно быть следующего вида: <Имя пользователя>.xml
    Формат данных профиля (файла данных) должен быть следующий:

<?xml version=“1.0” encoding=“Windows-1251”?>
<assistantData version=“0.1”>
<remins>
<remind id=“123”>
<title>Title</title>
<dateTime>01.02.2007 11:22:33</dateTime>
</remind>
</reminds>
<tasks>
<task id=””>
<title>заголовок</title>

<dateEnd>07.03.2007 11:22:33</dateEnd>
<remindBegin>True</remindBegin>
<remindEnd>True</remindEnd>
<remind>True</remind>
<remindInterval>600</remindInterval>
</task>
</tasks>
</assistantData>
