AIAssisant Virtual Engine

(Движок AIAssistant)

AIAssisant Virtual Engine (AIAssisant Engine) состоит из виртуальной машины (AIAsssitant Virtual Machine) и виртуальной среды обитания агентов (AIAssistant Virtual Enviroment).

Виртуальная среда AIAssistant предназначена для обеспечения работы компонентов системы AIAssistant. Виртуальная среда содержит в себе информацию об агентах, объектах, классов, связях и т.д. Агенты являются основными компонентами системы. Остальные компоненты служат для расширения умений агентов. Все компоненты системы написаны на специальном языке AIL. Данные между компонентами передаются в виде ACL сообщений.

Виртуальная машина AIAssistant предназначена для выполнения инструкций (программного кода) агентов и компонентов. Виртуальная машина создает и следит за работой подпроцессов (Thread) интерпретации и выполнения программного кода агентов (Code). Программный код всех компонентов реализуется на языке AIL.
